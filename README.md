# Jocul 21

O aplicație desktop dezvoltată în **C# (Windows Forms)** care simulează clasicul joc de cărți Blackjack ("21").

> **Notă:** Acest proiect a fost realizat în anul **2023** ca proiect de **atestat** profesional la informatică, la finalul clasei a XII-a.

## Descriere

Aplicația oferă o interfață grafică interactivă pentru jocul de Blackjack. Utilizatorul joacă împotriva calculatorului (Dealer-ul), scopul fiind de a obține un punctaj cât mai apropiat de 21, fără a-l depăși.

### Caracteristici principale:
*   **Meniu Principal:** Interfață de start prietenoasă.
*   **Gameplay:** Funcționalitate completă de trage carte (Hit) sau stai (Stand).
*   **Reguli:** Secțiune dedicată unde sunt explicate regulile jocului.
